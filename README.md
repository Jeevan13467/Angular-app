# Angular-app
## AWS EC2 Instance

- Go to AWS Console
- Instances(running)
- Launch instances

Pre-Requisites:
- Install nodejs 
- Install angular cli

### Run the below commands to install nodejs  and Angular cli

Install Nodejs 

```
sudo apt update
sudo apt install -y nodejs npm
```
check version 
```
node -v
npm -v
```
Install Nodejs 
```
sudo npm install -g @angular/cli
```
